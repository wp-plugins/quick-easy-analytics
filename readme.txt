== Description ==

Uses Google's newer Universal Analytics (analytics.js) tracking method, as opposed to the classic ga.js method. Provides standard pageview tracking on all front end content.

* Once plugin is installed, visit Settings > Google Analytics
* Enter the [UA number](https://support.google.com/analytics/answer/1032385) for your Google Analytics property and save.
* The tracking code will now be added to your site header.


== Installation ==

1. Do one of the following:

  1. Find and install the plugin from your site's plugin browser.

  1. Download `quick-easy-google-analytics.zip` from wordpress.org and upload the zip file 

  1. Upload `quick-easy-google-analytics` to the `/wp-content/plugins/` directory

1. Activate the plugin.

1. Follow the instructions in the Description to add the tracking code to your site.
